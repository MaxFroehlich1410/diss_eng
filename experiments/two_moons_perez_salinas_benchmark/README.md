# Perez-Salinas Benchmark

## Overview

Benchmark the Perez-Salinas data-reuploading model on two-moons.

## Run

```bash
python -m experiments.two_moons_perez_salinas_benchmark.run
python -m experiments.two_moons_perez_salinas_benchmark.build_figures
```

## Outputs

Generated artifacts are written under `results/` and are ignored by git.
