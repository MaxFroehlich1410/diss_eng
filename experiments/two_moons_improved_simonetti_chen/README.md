# Improved Simonetti Chen Benchmark

Benchmark improved Simonetti and Chen variants against the shared optimizer suite.

## Run

```bash
python -m experiments.two_moons_improved_simonetti_chen.run
python -m experiments.two_moons_improved_simonetti_chen.build_figures
```
