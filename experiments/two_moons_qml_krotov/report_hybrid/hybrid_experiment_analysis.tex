\section*{Experimental setup}
This report benchmarks four optimizer families on quantum machine learning classification tasks:
\begin{itemize}
\item \textbf{Hybrid Krotov}: online stale-adjoint update for an initial phase, then switching to the full-batch Krotov update at iteration 10.
\item \textbf{Adam} with full-batch parameter-shift gradients.
\item \textbf{L-BFGS-B} with full-batch parameter-shift gradients.
\item \textbf{Quantum Natural Gradient (QNG)}: full-batch BCE gradient preconditioned by the Fubini-Study metric tensor, recomputed at every iteration, with Tikhonov regularisation $\lambda$.
\end{itemize}

Three experiments are reported. The first two use literature-derived QML models on the two-moons classification task:
\begin{itemize}
\item \textbf{Simonetti--Perri--Gervasi hybrid classifier}: full hybrid model including the classical readout head, 1000 two-moons samples, noise 0.05, 20 iterations, 3 seeds.
\item \textbf{Chen SUN-VQC}: two macro-layers, richer $(R_y,R_z)$ data encoding, 300 two-moons samples, noise 0.07, 20 iterations, 3 seeds.
\end{itemize}
The third experiment uses a 4-qubit, 3-layer HEA on the binary Iris dataset (100 iterations, 5 seeds), described in a later section.

For each model, all optimizers see the same train/test split, the same random seed, the same iteration budget, and the same full trainable parameter vector. The hybrid Krotov implementation updates gate-supported parameters via the Krotov contribution while non-gate parameters (such as the Simonetti classical head) receive exact loss gradients within the same epoch. The fair accounting axis remains
\[
\text{cost units} = \text{sample forward passes} + \text{sample backward passes}.
\]

\section*{Main findings}
\begin{enumerate}
\item Under the fair benchmark, \emph{all four optimizers produce acceptable or excellent train/test results on both two-moons models}. Mean final test accuracies range from 0.906 (Adam on Chen) to 1.000 (L-BFGS-B and QNG on Simonetti), so the remaining differences are meaningful optimizer effects.
\item On the Simonetti model, QNG achieves the second-lowest final loss (0.134) after L-BFGS-B (0.001), substantially outperforming both hybrid Krotov (0.448) and Adam (0.372), while matching L-BFGS-B's perfect test accuracy.
\item On the Chen SUN-VQC, QNG (loss 0.399) and L-BFGS-B (0.393) finish in a near-tie, both ahead of hybrid Krotov (0.437) and Adam (0.484). QNG achieves the highest test accuracy (0.989).
\item Hybrid Krotov consistently shows the strongest first step across both models, confirming that the online-to-batch schedule makes an aggressive early move. On Chen, the hybrid-over-Adam gap persists through the full training budget.
\item On Iris, all four methods converge to strong classifiers. Hybrid Krotov achieves the lowest final loss and the fastest threshold crossing.
\end{enumerate}

\section*{Iris binary classification benchmark}
To test whether the optimizer ranking generalizes beyond the two-moons geometry, we run an additional benchmark on the Iris dataset restricted to a binary task (setosa vs.\ versicolor, first two features, scaled to $[0,\pi]$). The circuit is the same 4-qubit, 3-layer HEA used in the original two-moons study. The protocol uses 5 random seeds, 100 outer iterations, and the same fair cost accounting as before.

On Iris all four optimizers reach the $\leq 0.40$ threshold in every seed, and all achieve strong final test accuracy ($\geq 0.97$). The main differentiator is speed. Hybrid Krotov crosses the threshold fastest (0.3\,s on average, after just one online update), followed by L-BFGS-B (0.9\,s), QNG (2.4\,s), and Adam (3.4\,s). On final loss, hybrid Krotov is again best (0.255) and has zero tail oscillation, matching L-BFGS-B.  QNG converges to a comparable final loss (0.267) and shows very low tail noise (0.0001), but at substantially higher wall-clock cost (39.7\,s vs.\ 18.6\,s for hybrid Krotov) because of the per-iteration metric-tensor overhead.

\section*{Conclusion}
This report benchmarks hybrid Krotov against Adam, L-BFGS-B, and QNG across three experiments: the Simonetti full hybrid model on two-moons, the Chen SUN-VQC on two-moons, and a 4-qubit HEA on binary Iris.

\textbf{Simonetti (two-moons).} All four optimizers train the model extremely well. L-BFGS-B dominates with near-zero loss; QNG finishes second with loss 0.134 and perfect test accuracy; Adam and hybrid Krotov produce slightly weaker objective values but still excellent classification.

\textbf{Chen SUN-VQC (two-moons).} QNG and L-BFGS-B finish in a near-tie (0.399 vs.\ 0.393), both ahead of hybrid Krotov (0.437). QNG achieves the highest test accuracy (0.989). Hybrid Krotov retains a clear advantage over Adam on all metrics. The hybrid-over-Adam signal survives the addition of the stronger QNG baseline.

\textbf{Iris HEA.} All four optimizers converge to strong classifiers. Hybrid Krotov achieves the lowest final loss and the fastest threshold crossing. QNG, with a properly tuned learning rate, reaches comparable final performance but at substantially higher computational cost.

Taken together, the evidence shows that QNG is a strong optimizer when properly tuned, achieving low loss and high accuracy across all models, though at higher computational cost due to the per-iteration metric-tensor overhead. Hybrid Krotov remains competitive through its aggressive early convergence and stable late-stage refinement, particularly on gate-centric circuits. L-BFGS-B remains the strongest tested method on the two-moons models whenever second-order information is cheap to obtain. Adam is consistently the weakest of the four on these benchmarks, though it still produces acceptable classifiers in all cases.
