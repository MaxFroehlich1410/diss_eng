# Two-Moons Krotov Variants

## Overview

Compare online, batch, minibatch, and hybrid Krotov variants.

## Run

```bash
python -m experiments.two_moons_krotov_variants.run
python -m experiments.two_moons_krotov_variants.build_report
```

## Outputs

Generated artifacts are written under `results/` and are ignored by git.
