# Two-Moons Baseline

## Overview

Baseline two-moons benchmark comparing hybrid Krotov against gradient-based baselines.

## Run

```bash
python -m experiments.two_moons_baseline.run
python -m experiments.two_moons_baseline.plot_results
python -m experiments.two_moons_baseline.analyze_results
python -m experiments.two_moons_baseline.build_report
```

## Outputs

Generated artifacts are written under `results/` and are ignored by git.
